Bioinformatics Workflows

RNA‑seq Workflow

FASTQ → Quality Control (FastQC) → Alignment (STAR/HISAT2) → Counting (featureCounts) → Differential Expression (DESeq2) → Visualization (R/Python).

Proteomics Workflow

MS Raw Data → MaxQuant Processing → Protein Identification → Quantification → Functional Analysis.

Metabolomics Workflow

LC‑MS Data → Peak Detection (XCMS) → Alignment → Statistical Analysis → Pathway Interpretation.

Multi‑Omics Integration

Combine genomics, transcriptomics and proteomics data using tools like mixOmics, WGCNA and MOFA2.
